# Contributing

Thanks for your interest in contributing to the Desmos VS Code Extension!

## Getting Started

1. Fork and clone the repository
2. Install dependencies: `npm install`
3. Open the project in VS Code
4. Press `F5` to launch the Extension Development Host for testing

## Making Changes

1. Create a new branch: `git checkout -b my-feature`
2. Make your changes
3. Run tests: `npm test`
4. Commit your changes with a descriptive message
5. Push to your fork and open a Pull Request

## Reporting Bugs

Open an issue on [GitHub](https://github.com/evan-kolberg/desmos-vscode-extension/issues) with:

- Steps to reproduce
- Expected vs actual behavior
- VS Code version and OS

## Code Style

- Keep it simple and readable
- Follow the existing patterns in the codebase
- Test your changes before submitting
